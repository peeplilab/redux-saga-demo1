# react-redux-saga

The project was generated from `create-react-app` and then I included a few libraries.

## Features Out-Of-The-Box

* React-Router 4
* Material Ui
* Redux
* Redux Saga
* ESlint
* Airbnb's ESlint rules


## Installation

Clone repo and run:

```
yarn && yarn start
```
